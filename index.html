Format Guide

MC Multiple Choice

// Asterisk marks the ONE correct answer

What is 2 + 2?
a) 3
*b) 4
c) 5
d) 6

MA Multiple Answer

// Asterisk marks ALL correct answers
// (two or more asterisks = multiple answer)

Which are primary colors?
*a) Red
b) Green
*c) Blue
*d) Yellow
e) Purple

T/F True / False

// Choices must be True and False

The sky is blue.
*a) True
b) False

MT Matching

// Pairs separated by =
// No asterisk needed

MATCHING: Match each term.
a) Photosynthesis = Plants making food
b) Respiration = Cells releasing energy
c) Osmosis = Water crossing membranes

FB Fill in the Blank

// Use FITB: prefix
// Correct answer on next line with *

FITB: The capital of France is _____.
*Paris

ES Essay

// Use ESSAY: prefix

ESSAY: Explain the impact of
globalization on rural communities.

Canvas Import Steps

1Go to Settings → Import Course Content
2Set Content Type to QTI .zip file
3Leave "Convert to New Quizzes" unchecked
4Upload your .zip file and click Import
5Find your quiz under Quizzes in the course
6To use as a bank: open the quiz, click Settings → Build Question Bank
Canvas guide: Import quizzes from QTI packages ↗
Option A — Import as a Quiz
Questions land directly in a new New Quizzes assignment, ready to assign to students.
1In your course, go to Settings → Import Course Content
2Set Content Type to QTI .zip file
3Check "Convert content to New Quizzes"
4Upload your .zip file and click Import
5Find the new quiz under Quizzes in your course
Canvas guide: Import a quiz from a QTI package ↗
Option B — Import into an Item Bank
Questions go into a reusable bank. Draw from the bank when building any New Quiz in the course.
1In your course, go to Quizzes and open or create a New Quiz
2Click Item Banks in the quiz editor
3Open an existing bank or create a new one
4Click the bank's options menu (⋮) and select Import
5Upload your .zip file — questions will be added to the bank
6Return to your quiz and insert questions from the bank
